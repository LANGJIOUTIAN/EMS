## [1.0.0] - 2026-01-27

### ✨ 新增功能

- 🖥️ **CPU/GPU 监控模块**
  - 实时使用率显示
  - 温度监控
  - 内存使用统计
  - 可视化进度条

- 🌐 **网络监听模块**
  - 流量统计
  - 连接数管理
  - 上传/下载速率显示
  - 异常检测

- 📱 **软件行为监控模块**
  - 进程监控
  - 网络请求追踪
  - 文件操作记录
  - 屏幕监控状态

- 🔌 **串口监听模块**
  - 端口状态监控
  - 数据速率显示
  - 设备检测
  - 异常帧计数

- 🌀 **风扇控制模块** ⭐
  - 自动调节模式
  - 手动控制模式
  - 滑块精确控制（0-100%）
  - 实时 RPM 显示
  - 多风扇独立管理

- 📊 **数据可视化**
  - 性能趋势图
  - 网络拓扑图
  - ECharts 集成
  - 实时数据更新

- ⚠️ **告警系统**
  - 多级告警（紧急/警告/信息）
  - 告警聚合
  - 时间戳记录
  - 模块标识

- 🎨 **科技感 UI**
  - 深色主题
  - 霓虹光效
  - 粒子动态背景
  - 扫描线效果
  - 数字滚动动画

- 🖥️ **Electron 桌面应用**
  - 系统托盘
  - 开机自启
  - 后台运行
  - IPC 通信

### 📦 独立 Web 版本

- 无需安装任何依赖
- 双击即用
- 跨平台兼容
- 完整功能演示

### 📚 完整文档

## 版本说明

### 语义化版本号

格式：`MAJOR.MINOR.PATCH`

- `MAJOR`：重大变更，可能不兼容
- `MINOR`：新增功能，向下兼容
- `PATCH`：Bug 修复，向下兼容

### 变更类型

- ✨ `新增功能` (feat)
- 🔧 `改进` (refactor)
- 🐛 `修复` (fix)
- 📚 `文档` (docs)
- 🎨 `样式` (style)
- ⚡ `性能` (perf)
- 🔬 `测试` (test)
