## 快速入门

### 第一次使用

1. **打开系统**
   - Web 版：用浏览器打开 `examples/standalone/unified-monitoring.html`
   - 桌面版：运行安装好的应用程序

2. **查看界面**
   - 顶部：系统信息和运行时间
   - 中部：各监控模块卡片
   - 底部：性能图表和告警中心

3. **实时数据**
   - 数据每 1.5 秒自动刷新
   - 部分模块有数字滚动动画效果

## 监控模块使用

### 🖥️ CPU/GPU 监控

#### 功能说明

该模块显示系统的 CPU 和 GPU 状态：

- **CPU 使用率**：0-100% 实时显示，进度条可视化
- **GPU 使用率**：0-100% 实时显示，紫色进度条
- **系统温度**：摄氏度显示
- **内存使用**：当前使用的 GB 数

#### 使用方法

1. 查看监控卡片左上角的图标和标题
2. 观察实时数值和进度条
3. 数值超过阈值会变色：
   - 绿色（正常）< 60%
   - 黄色（警告）60-80%
   - 红色（危险）> 80%

### 🌐 网络监听

#### 功能说明

- **总流量**：当前网络总流量（MB）
- **活动连接**：当前活跃网络连接数
- **上传速率**：上传速度（KB/s）
- **下载速率**：下载速度（KB/s）
- **异常检测**：检测到的网络异常数

### 📱 软件行为

#### 功能说明

- **运行进程**：当前运行的进程总数
- **异常行为**：检测到的软件异常
- **屏幕监控**：屏幕监控状态
- **网络请求**：软件发起的网络请求数
- **文件操作**：文件操作计数

### 🔌 串口监听

#### 功能说明

- **活跃端口**：当前打开的串口数量
- **数据速率**：串口传输速率（bps）
- **异常帧**：错误的数据包数量
- **心跳状态**：设备连接状态
- **设备数量**：连接的设备数

## 🌀 风扇控制

### 功能特点

- 自动模式：系统根据温度自动调节
- 手动模式：精确控制转速（0-100%）
- 多风扇管理：支持多个风扇独立控制
- RPM 显示：实时显示转速

### 自动模式

1. 在风扇控制卡片中，点击"自动"按钮
2. 按钮变为蓝色激活状态
3. 风扇根据系统温度自动调节转速
4. 无需手动干预

### 手动模式

1. 点击"手动"按钮
2. 按钮变为紫色激活状态
3. 拖动滑块来调整转速：
   - 0%：风扇停止（静音）
   - 50%：平衡模式
   - 100%：最大性能（最大噪音）
4. 滑块右侧显示当前设置值
5. RPM 实时更新显示

### 风扇信息卡片

每个风扇都有独立的信息卡片，显示：

- 风扇名称（如 "CPU 风扇"）
- 实时转速（RPM）
- 当前模式（自动/手动）

## ⚠️ 告警系统

### 告警级别

| 级别 | 颜色 | 说明 |
|------|------|------|
| 紧急 | 红色 | 需要立即处理的问题 |
| 警告 | 黄色 | 需要关注的问题 |
| 信息 | 蓝色 | 一般通知信息 |

### 告警卡片结构

```
[告警级别]  告警内容
            YYYY-MM-DD HH:MM:SS | 模块: XXX
```

### 告警特性

- **自动聚合**：相同类型的告警在 5 分钟内自动合并
- **自动过期**：只显示最近的 8 条告警
- **快速识别**：通过颜色快速识别级别
- **模块标识**：显示告警来自哪个模块

### 示例告警

- "CPU使用率过高，请关注"（CPU/GPU模块）
- "检测到异常网络连接"（网络模块）
- "GPU温度升高"（CPU/GPU模块）
- "新设备接入网络"（网络模块）
- "串口数据异常"（串口模块）

## 📊 性能趋势图

### 图表类型

1. **CPU 趋势**：青色线条
2. **GPU 趋势**：紫色线条
3. **网络趋势**：绿色线条

### 图表功能

- 显示最近 60 秒的数据
- 鼠标悬停显示详细数值
- 可以点击图例切换显示/隐藏
- 支持窗口大小自适应

## ⚙️ 系统设置

### 可配置项（Electron 版）

在设置页面中可以配置：

| 选项 | 说明 | 默认值 |
|------|------|--------|
| 开机自启 | 系统启动时自动运行 | 关闭 |
| 后台运行 | 关闭窗口后继续运行 | 开启 |
| 主题 | 深色/浅色主题 | 深色 |
| 语言 | 界面语言 | 中文 |
| 通知 | 系统通知 | 开启 |

## 🎨 界面操作

### 鼠标操作

## 💡 使用技巧

### 1. 快速查看状态

看顶部状态栏：

- 运行时间：系统已运行时间
- 当前时间：实时时钟
- 在线状态：显示为绿色圆点

### 2. 风扇控制建议

- 日常使用：自动模式
- 游戏/渲染：手动 70-100%
- 夜间/静音：手动 0-30%

### 3. 告警处理

看到红色告警时：

1. 查看告警内容
2. 检查相关模块
3. 根据需要采取行动
4. 告警会自动滚动显示
