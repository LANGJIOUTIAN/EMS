## 安装相关

### Q: 需要安装 Node.js 才能使用吗？

**A:** 不需要！我们提供了独立 Web 版本：

- **独立 Web 版**：不需要 Node.js，双击即可用（推荐）
- **Electron 桌面版**：需要 Node.js（功能最全）

---

### Q: 支持哪些操作系统？

**A:** 支持以下系统：

- **Windows**：10 及以上版本
- **macOS**：10.14 及以上版本
- **Linux**：Ubuntu 18.04 及以上版本

Web 版本在所有支持现代浏览器的系统上都能运行。

---

### Q: npm install 失败怎么办？

**A:** 尝试以下步骤：

```bash
# 1. 清理缓存
npm cache clean --force

# 2. 删除旧文件
rm -rf node_modules package-lock.json

# 3. 使用淘宝镜像
npm install --registry=https://registry.npmmirror.com

# 4. 或者使用 yarn
yarn install
```

## 使用相关

### Q: 独立 Web 版的数据是真实的吗？

**A:** 独立 Web 版使用模拟数据，用于演示和测试。要获取真实的硬件数据，需要使用 Electron 桌面版。

---

### Q: 如何导出监控数据？

**A:** 

- **Web 版**：截图或手动记录（后续版本将支持导出）
- **Electron 版**：在设置中找到"导出数据"选项
- **数据格式**：支持导出为 Excel 和 PDF 格式

---

### Q: 可以在手机上使用吗？

**A:** 可以！

- Web 版完全响应式，手机浏览器可以正常使用
- 建议在横屏模式下获得最佳体验
- Electron 版不支持移动设备

---

### Q: 数据更新频率可以调整吗？

**A:** 可以！

- 默认：1.5 秒刷新一次
- 配置位置：`config/performance.js` 中的 `refreshInterval`
- 可调整范围：500ms - 10000ms（0.5秒 - 10秒）

---

## 功能相关

### Q: 风扇控制在 Web 版能用吗？

**A:** Web 版有风扇控制界面，但只能控制模拟数据。

- **Web 版**：有 UI，功能完整，但控制的是模拟的风扇
- **Electron 版**：可以真实控制硬件风扇（需要管理员权限）

---

### Q: 支持多少个风扇同时控制？

**A:** 理论上支持无限个！

- 自动检测系统中的所有风扇
- 每个风扇有独立的控制卡片
- 可以单独或批量控制

---

### Q: 告警通知会弹出吗？

**A:** 是的：

- **Web 版**：在浏览器内显示告警
- **Electron 版**：可以发送系统通知（在设置中启用）
- **通知方式**：桌面弹窗、声音（可选）

---

### Q: 可以自定义告警规则吗？

**A:** 可以！

- 在 `config/alerts.js` 中自定义阈值
- 可以设置警告和紧急两个级别
- 可以选择要监控哪些指标

## 技术相关

### Q: 项目是什么技术栈？

**A:**

- **前端**：HTML5, CSS3, JavaScript, ECharts
- **桌面应用**：Electron
- **硬件访问**：systeminformation 库
- **构建工具**：electron-builder

---

### Q: 安全吗？会上传我的数据吗？

**A:** 安全！

- 所有数据存储在本地
- 不会上传到任何服务器
- 网络模块仅供分析本地网络连接
- 代码开源，可自行审核

---

### Q: 可以修改 UI 颜色吗？

**A:** 可以！

- 在配置中自定义主题色
- 支持深色/浅色主题切换
- Web 版可以直接在浏览器开发者工具中实时修改

---

### Q: Electron 应用有签名吗？

**A:** 当前版本没有签名，您会看到安全警告。

- Windows：点击"更多信息" -> "仍要运行"
- macOS：需要在安全设置中允许
- 生产环境建议进行代码签名
