# 快速开始指南

## 🚀 30 秒快速启动

### 方式一：独立 Web 版（零依赖）

**Windows:**
```
1. 进入 `examples/standalone/` 目录
2. 双击 `启动监控系统.bat`
3. 浏览器会自动打开

**Mac/Linux:**
```
1. 进入 `examples/standalone/` 目录
2. 用浏览器打开 `unified-monitoring.html`
```

就这么简单！无需安装任何软件！
